MLP Classifier Prediction

This application uses a PyTorch MLP classifier trained with MLflow. Enter 20 comma-separated feature values to get a prediction.

The model classifies input into one of two classes (Class 0 or Class 1).

Example:

0.5, 1.2, -0.3, 0.8, 1.5, -0.7, 0.2, 0.9, -1.1, 0.4, 0.6, -0.5, 1.0, -0.2, 0.3, 0.7, -0.9, 1.3, -0.4, 0.1
